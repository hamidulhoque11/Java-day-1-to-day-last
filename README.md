Escape sequence-

         1. \b - Backspace
            
            2. \t - Tab
            
            3. \n - New Line
            
            4. \r - Carriage Return 
            
            5. \' - Single Quote 
            
            6. \" - Double quote
            
            7. \\ - Backslash
